# FlowLog - Sistema de Gestão de Estoque

Sistema desenvolvido em Python para controle de movimentação de produtos, focado em lógica de inventário e integração com banco de dados relacional.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **Banco de Dados:** MySQL
* **Versionamento:** Git & GitHub

## 🚀 Funcionalidades (Roadmap)
- [ ] Conexão segura com banco de dados.
- [ ] Cadastro de produtos com SKU único.
- [ ] Registro de entradas e saídas.
- [ ] Alerta de estoque baixo.

## 📦 Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt`.
3. Configure o banco no arquivo `config.py`.
4. Execute: `python src/main.py`.
